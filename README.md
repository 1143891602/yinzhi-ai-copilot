## 🎨 V3.0 核心升级：白色生命感 (White Life)

### 🍃 极致通透的视觉体系
- **从深色向“光”进化**：全面拥抱白色、雾感、柔发光的层级体系。
- **环境氛围光 (Ambient Glow)**：全局动态光晕随鼠标位置微调，界面如同在呼吸，而非静态表单。
- **极简玻璃态 (Premium Glass)**：采用 28px 超大模糊半径的背板材质，营造通透且专注的教研氛围。

### 🧠 更具仪式感的 AI 交互
- **智能构思态**：教案生成中新增“AI 正在构思音乐细节...”指示器，增强人机交互的生命感。
- **转场艺术 (Liquid Motion)**：全站路由跳转引入深度模糊与缩放复合动效，确保交互如乐句般流畅。

## 🚀 核心功能特性

### 🎹 智能教案生成 (LessonPlan)
- **RAG 知识增强**：AI 自动检索并结合您在「教研矩阵」中投喂的私有教学经验，生成最具机构特色的教案。
- **多格式导出**：一键导出高清 PDF 教案，适配线下教学与家长展示。

### 📊 学员成长档案 (Students)
- **五维能力雷达**：可视化展示学生在技术、乐感、节奏等维度的成长曲线。
- **定制化教学大纲**：基于学员当前能力分布，智能建议下一阶段的教学重点。

### 📡 家校沟通桥梁 (Communication)
- **多重人格模型**：支持“亲切、专业、简洁”三种风格，一键生成具有针对性的课后反馈。

## 🛠️ 全功能矩阵 (Feature Matrix)

| 功能模块 | 核心能力 (Core Capabilities) | 业务价值 (Business Value) |
| :--- | :--- | :--- |
| **仪表盘 (Dashboard)** | 动态趋势图表 (Recharts)、实时教研日志、全站数据指标概览 | 数字化呈现机构教研产能，实时把控教研进度 |

| **教研矩阵 (Research)** | 知识片段碎片化注入、教研资料批量解析、分类矩阵管理 | 建立机构专属的“教研大脑”，让 AI 越教越聪明 |
| **家校沟通 (Comm.)** | 多人格 AI 反馈模型 (亲切/专业/简洁)、一键复制成功反馈 | 极大地降低老师课后沟通成本，提升家长满意度 |
| **学员管理 (Students)** | 五维能力雷达模型、成长档案追踪、学习状态可视化 | 实现科学化带教，让每位学员的进步“有据可查” |
| **系统设置 (Settings)** | API 接口自由配置、数据一键本地备份、多模型引擎切换 | 确保数据隐私与接口灵活性，适配不同 AI 算力需求 |

## 🛠️ 技术栈 (Tech Stack)

- **核心框架**: React 18 + TypeScript + Vite 4
- **动效系统**: Framer Motion (Liquid Motion / Page Transitions)
- **数据可视化**: Recharts (Area / Bar / Radar Chart)
- **编辑器**: Tiptap (StarterKit / Underline / Highlight)
- **打包模式**: **Single-File HTML** (打包后仅 ~2.0MB，开箱即用)
