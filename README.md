# 音智 AI · 智能教研副驾驶 (V3.0 White Life / Vitality)

> 专为全国音乐类艺培机构打造的 **AI 智能教研副驾驶系统**。V3.0 版本带来了全新的 **“白色生命感” (White Life / Vitality)** 设计系统，通过 **环境氛围光、呼吸感交互、极简玻璃态**，为教研工作带来极致通透、专业且有温度的数字化体验。

## 🎨 V3.0 重大升级：白色生命感 (White Life)

### 1. 🍃 极致通透的视觉体系 (Vitality Design)
- **从深色向“光”进化**：告别高对比度的深色模式，全面拥抱白色、雾感、柔发光的层级体系。
- **环境氛围光 (Ambient Glow)**：引入随鼠标位置微调的全局动态光晕，让界面如同在呼吸，而非静态表单。
- **极简玻璃态 (Premium Glass)**：采用 28px 超大模糊半径的背板材质，配合点阵网格背景，营造通透的教研氛围。

### 2. 🧠 更具仪式感的 AI 交互 (AI Presence)
- **智能构思态**：教案生成过程中，新增“AI 正在构思音乐细节...”的打字指示器，增强人机交互的生命感。
- **场景化文案**：全站文案深度适配音乐教研场景，清理所有 Lorem Ipsum 占位符，统一为温润、专业的教研语言。
- **丝滑转场 (AnimatePresence)**：全站路由跳转引入深度模糊与缩放复合动效，确保每一次切换都如音乐乐句般流畅。

### 3. 🛠️ 交互与稳定性增强
- **交互连通性**：Dashboard 日志直接链接教案库，教案库参数一键回流生成器，打破功能孤岛。
- **反馈闭环**：家校沟通页新增真实的剪贴板复制成功反馈，教研库上传增加模拟解析动画。
- **性能飞跃**：优化了 Recharts 图表渲染与 Framer Motion 的布局计算，低配设备亦能流畅运行。

## 🚀 核心功能特性

### 1. 🎹 智能教案生成 (LessonPlan)
- **多维度配置**：支持乐器种类、学生级别、课题主题的快速选择。
- **RAG 知识增强**：AI 会自动检索并结合你投喂在「教研矩阵」中的私有教学经验。
- **多格式导出**：支持高清 PDF 导出，方便打印或直接分发。

### 2. 📡 家校沟通桥梁 (Communication)
- **多重人格模型**：支持“亲切、专业、简洁”三种沟通风格切换。
- **一键触达家长**：自动生成暖心、具有针对性的课后反馈，大幅节省沟通成本。

### 3. 🗃️ 教研矩阵/知识库 (TeachingResearch)
- **教学片段注入**：支持录入核心技巧、考级大纲。
- **文件智能解析**：支持 PDF/DOCX 格式教研资料的批量上传（模拟）。

### 4. 📊 学员档案管理 (Students)
- **五维能力雷达**：可视化展示学生在音准、节奏、表现力等维度的成长曲线。
- **智能进度追踪**：记录课时余额、带教老师及下次考级计划。

## 🛠️ 技术栈 (Tech Stack)

| 维度 | 技术实现 |
|------|---------|
| **核心框架** | React 18 + TypeScript + Vite 4 |
| **动效系统** | Framer Motion (Liquid Motion / Page Transitions) |
| **图标库** | Lucide React |
| **编辑器** | Tiptap (StarterKit / Underline / Highlight) |
| **数据可视化** | Recharts (Area / Bar / Radar Chart) |
| **PDF 导出** | jsPDF + html2canvas |
| **打包模式** | **Single-File HTML** (打包后仅 ~2.0MB) |

## 📦 快速部署

1. **直接体验**：下载 `音智AI教研系统.html`，使用 Chrome 浏览器直接双击打开即可使用。
2. **API 配置**：在「系统设置」中填入您的 OpenAI 兼容接口地址与 Key（推荐配合 `proxy-server.mjs` 使用）。
3. **本地开发**：
   ```bash
   npm install
   # 启动跨域代理
   node proxy-server.mjs &
   # 启动开发服务
   npm run dev
   ```
